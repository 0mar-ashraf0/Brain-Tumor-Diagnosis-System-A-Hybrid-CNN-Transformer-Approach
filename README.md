# Brain-Tumor-Diagnosis-System-A-Hybrid-CNN-Transformer-Approach
SOTA Brain Tumor Diagnosis using a Hybrid EfficientNet + Swin Transformer architecture. Features Explainable AI (Grad-CAM) for heatmap visualization, weighted loss for class imbalance, and active learning for edge cases. Classifies Glioma, Meningioma, Pituitary, and No Tumor with high accuracy. Deployed via Gradio.
